Connect 4 - Monte Carlo Game Player
[CODE]

Monte Carlo AI agent designed to play connect 4 game.
The AI agent runs a monte carlo tree search on the current game state to decide the next best move.
Difficulty settings in the game will decide the number of simulations to perform.
Since connect-4 is a solved game, it is impossible to win the game in highest difficulty settings.
Movie Recommender System (Mathworks Hackathon Winner!)
[CODE] [BLOG PAGE]

Movie recommender system that is trained using thousands of film posters.
This application recommends movies based on an input image (image that best describes your interest).
A convolutional auto-encoder maps the input image to a lower dimentional latent space, and recommends all the other points(movies) close to the input.
